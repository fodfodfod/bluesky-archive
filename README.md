# Bluesky Archive
A tool to archive all posts and images from bluesky users

## Instalation
### Docker compose
Clone the repo.
Add usernames to `user_data/add_list.txt`


Run
```sh
docker compose up
```
### Install locally (Untested)
Clone the repo

Install python dependencies
```sh
pip install -r requirements.txt
```

Run the app
```sh
python main.py
```
