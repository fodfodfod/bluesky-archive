# Bluesky Archive
A tool to archive all posts and images from bluesky users

## Instalation
### Docker compose
Clone the repo.
Run
`docker compose up`
### Install locally (Untested)
Clone the repo

Install python dependencies
`pip install -r requirements.txt`

Run the app
`python main.py [username or did]`
